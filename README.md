# Random-Seats-of-G2011
- **Under Senior High**
- 本程序为QHFZ G2011 方便地随机抽取座位及按照固定方式轮换座位设计，可以应用于其他大部分场景
- 可执行文件可以在 Windows10 及更高版本运行，源码运行需 Python3.5 以上环境以及 Pygame、PyQt5 中的部分或全部第三方包
